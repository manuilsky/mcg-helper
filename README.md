# Инструкция по установке и обновлению MCG Helper


## 1. Установка расширения

1. Скачайте архив **`mcg-helper.zip`** из последнего релиза на странице [Releases](https://github.com/manuilsky/mcg-helper/releases).
2. Распакуйте скачанный архив в любую удобную папку на компьютере (например, в созданную папку `mcg-helper`).
3. Откройте Google Chrome и перейдите по адресу `chrome://extensions/` (или откройте **Меню** -> **Расширения** -> **Управление расширениями**).
4. В правом верхнем углу включите переключатель **Режим разработчика** (Developer mode).
5. В левом верхнем углу нажмите кнопку **Загрузить распакованное расширение** (Load unpacked) и выберите папку, в которую распаковали файлы (папка должна содержать файл `manifest.json`).

---

## 2. Обновление расширения

Для обновления до последней версии перейдите в папку с установленным расширением и запустите файл обновления:

* **Для Windows**: запустите файл **`update.bat`** двойным кликом.
* **Для Linux / macOS**: откройте терминал в папке расширения и выполните:
  ```bash
  chmod +x update.sh && ./update.sh
  ```

После завершения работы скрипта перейдите на страницу `chrome://extensions/` и нажмите иконку **Обновить** (круговая стрелка) на карточке расширения **MCG Helper**.
